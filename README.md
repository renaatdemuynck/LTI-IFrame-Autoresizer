# LTI IFrame Autoresizer
Automatically resizes the iframe of an LTI tool

## Installation

From NPM:

```sh
npm install lti-iframe-autoresizer
```

From Yarn:

```sh
yarn add lti-iframe-autoresizer
```

## Usage Example

```javascript
import LtiResizer from 'lti-iframe-autoresizer';

LtiResizer.init();
```
